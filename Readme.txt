Apple Catcher
Version 1.0

This game was made using phaser.js framework.

Created for Derivco Slice Of Source hackathon held at NMMU in Port Elizabeth.

You can use the keyboard arrow keys as a controller
Or you can use a leap motion as a controller.

Setting up leapmotion:
1. Plug leapmotion in.
2. Install Drivers for leapmotion.
3. Connect to the internet in order to download latest leap.js framework.
